Task Tracker - Simple Web Application
A lightweight, browser-based task management application to help you track your daily tasks.

Features
 Add Tasks: Quickly add new tasks with an intuitive input form

 View All Tasks: See all tasks in a clean, organized list

Mark as Completed: Toggle task completion status with a single click

Task Statistics: View counts for total, pending, and completed tasks

Delete Tasks: Remove tasks you no longer need

Local Storage: Your tasks are automatically saved in your browser

How to Use
Adding a Task
Type your task in the input box at the top

Click "Add Task" or press Enter

Completing a Task
Click the checkbox next to any task to mark it as completed

Click again to mark it as pending

Deleting a Task
Click the "Delete" button next to any task to remove it
